# AIReader
一个简易的阅读客户端，包含知乎日报，网易头条，妹子图片

#### 涉及的技术点：
1. Retrofit的使用;
2. RxJava的使用，包括和Retrofit的搭配;
3. MVP架构;
4. RecycleView + CardView 的使用;
5. 使用Picasso来加载图片;
6. WebView的使用，包括加载带css样式的文字;
7. 使用NavigationView进行菜单切换;
8. ButterKnife的使用;
9. CoordinatorLayout配合AppBarLayout和CollapsingToolbarLayout实现图片伸缩和工具栏;
10. FloatingActionButton和Snackbar的使用;
